𝗚𝗿𝗼𝘂𝗽𝗶𝗻𝗴

• {} - Inline group, all commands treated as one, must be terminated with a semi-colon at the final command.
• () - Subshell, all commands executed in a hidden shell.
Notes: If the parent process is killed before the child process, the child process is inherited by init, with a default PID of 1, known as a "Zombie Process". Variables in a subshell are only available inside of it (local variables).
• (()) - Arithmetic expression, for doing mathematical operations.
Notes: Can only perform integer math. If decimals or square roots are required, use the "bc" utility.

𝗔𝘀𝘀𝗼𝗰𝗶𝗮𝘁𝗶𝘃𝗲 𝗔𝗿𝗿𝗮𝘆

#Declare an associative array using the -A flag.
declare -A [IDENTIFIER]
#Add key-value pairs.
[IDENTIFIER][name]="John Doe"
[IDENTIFIER][email]="john@example.com"
[IDENTIFIER][age]=30
#Access and print a specific value using its key.
echo "Name: ${[IDENTIFIER][name]}"
echo "Email: ${[IDENTIFIER][email]}"
# Print all keys.
echo "Available information keys: ${![IDENTIFIER][@]}"
# Print all values.
echo "User details values: ${[IDENTIFIER][@]}"
# Iterate over all key-value pairs.
echo "Looping through all details:"
for key in "${![IDENTIFIER][@]}"; do
    echo "$key => ${[IDENTIFIER][$key]}"
done
